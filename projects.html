JavaScript Projects

The final project will be a website consisting of five to six individual pages. Each page will contain the following themes introduction, about, resume, projects and past work, and contact page. A CSS style sheet is required to control the layout and design of each page. The JavaScript pages will control the photo carousel and projects page. The first page will be the Index page, which is the introduction page. In this page, there will be a brief introduction and my mission statement and core values.

    • The layout of the About page controll by the basic1.css file, which controls the entire website.
    • The resume page is formatted with bullets in order to present the resume as it were in a printed document. A div class was created to style the in-line block bullets. The rest is a div to set the list of items
    • The Project page is controlled by the java.js and projects.js to activate the "Click Me". Once clicked the animated blue box will start on the upper right corner and slide diagnally to the center of the yellow box.
    • The contact page will contain contact information and controlled by the form.js file. The form.css file controlls the fields.
    • The CSS style sheet will control the layout and design flow of the entire website. It also contains styling feature unique to individual page and sections within pages.
    • A separate folder will be created as a central repository for all images. Each images will be referenced in this folder with the HREF tag.

The bouncing Twitter icon is set to bounce at determined rate and bounce count within the JavaScript in the projects.js file.
